PAGE FORMAT TAGS

Tag Name Description
<header> Header Where you place the contents of the page header.
<nav> Nav Used to specify the navigation bar.
<main> Main To separate the main body of the page from the header and footer.
<header> Header Description
<footer> Footer The section at the bottom of a web page usually used for contact information
<section> Section Defines elements in a document such as chapters headings or areas with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, commetnts etc.
<aside> Aside Contains information that is related to the main content but not essential for understanding.

MEDIA TAGS

Tag Name Description
<figure> Figure Encapsulates all types of media.
<figcaption> Figcaption Used to describe the media in <figcaption>.
<video> Video Used to embed videos.
<audio> Audio Used to embed ausio.
